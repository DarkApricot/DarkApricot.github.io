Winter Wonder Ride!

This is Project Wonder Winter Ride!

One of my biggest challenges (and eventually successes) of this project were the multiple slots. I've made those using only PlayerPrefs, since I had never even heard of .Json files at this point. ðŸ˜­

I used a system of saving strings with specific information to numbers associated with each file. For example, names were saved in a string with "nameFile" + fileNumber, cars with "whichCar" + "file" (nameFile + fileNumber), and so on.

This type of saving was particularly tricky to use when I made the TOP10, since I had no way to associate the names of files with the now sorted top10. To counter this I made a scriptable object, which was essentially just a manifestation of each file. I spawned them in with the correct visuals and named the objects the deltaTime float, so I could sort the objects on their names. Then all I had to do was put them in the UI in order of the sorted array.
